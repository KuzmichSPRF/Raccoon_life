# 🦝 Админ-команды для бота Raccoon Life

## Команды управления шишками

Все команды доступны **только администратору** (ADMIN_ID из .env).

---

### 💰 `/add <username|user_id> <amount> [reason]`

**Начисляет токены пользователю и отправляет уведомление**

#### Параметры:
- `username|user_id` — username (с @ или без) **ИЛИ** ID пользователя в Telegram (обязательно)
- `amount` — количество шишек для начисления (обязательно)
- `reason` — причина начисления (необязательно, по умолчанию "Начисление админом")

#### Примеры:
```
/add @kuzmi 100 За победу в турнире
/add kuzmi 500 Приз за первое место
/add 456789123 50 Бонус за активность
/add 123456789 100 За победу в турнире
```

#### Что происходит:
1. ✅ Шишки начисляются на баланс пользователя
2. 📬 Админ получает подтверждение с новым балансом
3. 📬 Пользователь получает уведомление в боте о начислении

---

### 💸 `/spend <username|user_id> <amount> [reason]`

**Списывает токены у пользователя и отправляет уведомление**

#### Параметры:
- `username|user_id` — username (с @ или без) **ИЛИ** ID пользователя в Telegram (обязательно)
- `amount` — количество шишек для списания (обязательно)
- `reason` — причина списания (необязательно, по умолчанию "Списание админом")

#### Примеры:
```
/spend @kuzmi 50 Штраф за читы
/spend kuzmi 100 Возврат ставки
/spend 123456789 50 Штраф за читы
```

#### Что происходит:
1. ✅ Шишки списываются с баланса пользователя
2. 📬 Админ получает подтверждение с остатком
3. 📬 Пользователь получает уведомление о списании

#### Ошибки:
- ❌ Если у пользователя недостаточно шишек, бот сообщит об этом

---

### 💳 `/balance <username|user_id>`

**Проверяет баланс пользователя**

#### Параметры:
- `username|user_id` — username (с @ или без) **ИЛИ** ID пользователя в Telegram (обязательно)

#### Примеры:
```
/balance @kuzmi
/balance kuzmi
/balance 123456789
```

#### Что показывает:
- 👤 Имя пользователя и username
- 🆔 ID пользователя
- 💰 Текущий баланс
- 📊 Всего заработано шишек
- 💸 Всего потрачено шишек

---

## 🔐 Безопасность

- Команды доступны **только администратору** (ADMIN_ID из .env)
- При попытке использования обычным пользователем бот ответит: "❌ У вас нет прав для этой команды!"

---

## 📝 Логи

Все операции записываются в лог бота:
- 🎯 Начисления: `💰 +{amount} Шишек: user_id={id}, reason=admin_grant:{reason}`
- 🎯 Списания: `💸 -{amount} Шишек: user_id={id}, reason=admin_spend:{reason}`
- 🎯 Уведомления: `📬 Уведомление отправлено пользователю {user_id}`

---

## ⚠️ Важные заметки

1. **Пользователь должен быть запущен в боте** хотя бы раз (использовать /start)
2. **Если пользователь заблокировал бота**, шишки всё равно будут начислены/списаны, но уведомление не дойдёт
3. **Username работает с @ и без** — `/add @kuzmi 100` и `/add kuzmi 100` равнозначны
4. **Поиск по имени** — если username не найден, бот попробует найти по first_name или last_name

---

## 🆘 Помощь

Если возникли проблемы:
1. Проверьте что ADMIN_ID настроен в .env
2. Проверьте логи бота на наличие ошибок
3. Убедитесь что пользователь существует (команда /balance @username)
