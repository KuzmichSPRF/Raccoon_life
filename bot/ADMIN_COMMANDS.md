# 🦝 Админ-команды для бота Raccoon Life

## Команды управления токенами

Все команды доступны **только администратору** (ADMIN_ID из .env).

---

### 💰 `/add <user_id> <amount> [reason]`

**Начисляет токены пользователю и отправляет уведомление**

#### Параметры:
- `user_id` — ID пользователя в Telegram (обязательно)
- `amount` — количество токенов для начисления (обязательно)
- `reason` — причина начисления (необязательно, по умолчанию "Начисление админом")

#### Примеры:
```
/add 123456789 100 За победу в турнире
/add 987654321 500 Приз за первое место
/add 456789123 50 Бонус за активность
```

#### Что происходит:
1. ✅ Токены начисляются на баланс пользователя
2. 📬 Админ получает подтверждение с новым балансом
3. 📬 Пользователь получает уведомление в боте о начислении

---

### 💸 `/spend <user_id> <amount> [reason]`

**Списывает токены у пользователя и отправляет уведомление**

#### Параметры:
- `user_id` — ID пользователя в Telegram (обязательно)
- `amount` — количество токенов для списания (обязательно)
- `reason` — причина списания (необязательно, по умолчанию "Списание админом")

#### Примеры:
```
/spend 123456789 50 Штраф за читы
/spend 987654321 100 Возврат ставки
```

#### Что происходит:
1. ✅ Токены списываются с баланса пользователя
2. 📬 Админ получает подтверждение с остатком
3. 📬 Пользователь получает уведомление о списании

#### Ошибки:
- ❌ Если у пользователя недостаточно токенов, бот сообщит об этом

---

### 💳 `/balance <user_id>`

**Проверяет баланс пользователя**

#### Параметры:
- `user_id` — ID пользователя в Telegram (обязательно)

#### Примеры:
```
/balance 123456789
/balance 987654321
```

#### Что показывает:
- 👤 Имя пользователя
- 💰 Текущий баланс
- 📊 Всего заработано токенов
- 💸 Всего потрачено токенов

---

## 🔐 Безопасность

- Команды доступны **только администратору** (ADMIN_ID из .env)
- При попытке использования обычным пользователем бот ответит: "❌ У вас нет прав для этой команды!"

---

## 📝 Логи

Все операции записываются в лог бота:
- 🎯 Начисления: `💰 +{amount} $ENT: user_id={id}, reason=admin_grant:{reason}`
- 🎯 Списания: `💸 -{amount} $ENT: user_id={id}, reason=admin_spend:{reason}`
- 🎯 Уведомления: `📬 Уведомление отправлено пользователю {user_id}`

---

## ⚠️ Важные заметки

1. **Пользователь должен быть запущен в боте** хотя бы раз (использовать /start)
2. **Если пользователь заблокировал бота**, токены всё равно будут начислены/списаны, но уведомление не дойдёт
3. **user_id можно узнать** через @userinfobot в Telegram или из логов бота

---

## 🆘 Помощь

Если возникли проблемы:
1. Проверьте что ADMIN_ID настроен в .env
2. Проверьте логи бота на наличие ошибок
3. Убедитесь что пользователь существует (команда /balance)
